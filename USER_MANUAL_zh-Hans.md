## 关于应用

**QuickMemo** 是一款快速备忘录应用，让您在想法出现的瞬间就能记录下来。

### 主要特点

- 📱 **摇动手势**：只需摇动设备即可立即打开备忘录输入界面
- 📁 **分类管理**：创建自定义分类，如工作、个人和想法
- 🏷️ **智能标签**：根据内容自动建议标签，支持多个标签进行详细分类
- 🤖 **AI功能**：标签提取、备忘录整理、分类摘要（需要API密钥）
- 📅 **日历集成**：自动将备忘录记录到日历
- ⌚ **Apple Watch支持**：直接在Watch上输入备忘录，支持离线使用
- 🔄 **数据同步**：Pro版本支持iCloud同步

## 核心功能

### 创建备忘录

#### 使用摇动手势创建
1. 轻轻摇动iPhone
2. 备忘录输入界面自动打开
3. 输入文本
4. 选择分类和标签（可选）
5. 点击"保存"

#### 从主屏幕创建
1. 点击底部的"+"按钮
2. 在当前选择的分类中创建备忘录
3. 输入文本并点击"保存"

### 编辑备忘录

1. 从列表中点击要编辑的备忘录
2. 编辑文本
3. 可以更改分类、标签或持续时间
4. 点击"更新"保存

### 删除备忘录

1. 在列表中向左滑动备忘录
2. 点击"删除"
3. 或者在编辑界面点击右上角的垃圾桶图标

### 搜索备忘录

1. 点击备忘录列表顶部的搜索栏
2. 输入关键词
3. 搜索标题、内容和标签

## 使用方法

### 管理分类

#### 创建分类
1. 打开设置
2. 点击"分类管理"
3. 使用"+"按钮添加新分类
4. 设置名称、图标和颜色

#### 编辑分类
1. 在分类管理中点击分类
2. 更改名称、图标或颜色
3. 设置默认标签

#### 重新排序分类
1. 在分类管理右上角点击"编辑"
2. 拖放更改顺序

#### 删除分类
1. 在分类管理中向左滑动分类
2. 点击"删除"
3. 该分类中的备忘录将移至"其他"

### 管理标签

#### 为备忘录添加标签
1. 在备忘录创建/编辑界面打开标签部分
2. 从分类的默认标签中选择
3. 或使用"新建"按钮添加新标签

#### 删除标签
1. 在分类编辑界面显示默认标签
2. 向左滑动标签删除

### 日历集成

#### 设置日历集成
1. 打开设置
2. 打开"日历集成"
3. 允许访问日历

#### 设置备忘录持续时间
1. 在备忘录创建/编辑界面打开"持续时间"部分
2. 选择15分钟、30分钟、1小时或2小时
3. 备忘录将作为日历事件记录

## AI功能

QuickMemo支持三个AI提供商：Gemini、Claude和ChatGPT。

### 设置AI功能

#### 配置API密钥
1. 打开设置
2. 点击"AI功能"
3. 输入所需AI提供商的API密钥
   - **Gemini**：用于标签提取
   - **Claude**：用于备忘录整理和摘要
   - **ChatGPT**：可用于所有功能

#### 如何获取API密钥
- **Gemini**：[Google AI Studio](https://makersuite.google.com/app/apikey)
- **Claude**：[Anthropic Console](https://console.anthropic.com/)
- **ChatGPT**：[OpenAI Platform](https://platform.openai.com/)

### 标签提取

AI自动从备忘录内容建议相关标签。

1. 打开备忘录创建/编辑界面
2. 输入文本（20个字符以上）
3. 点击"AI提取"按钮
4. 从建议的标签中选择并添加

**注意**：需要Gemini API密钥。

### 备忘录整理

AI将备忘录内容重新格式化为各种样式。

1. 在编辑界面点击备忘录整理按钮（魔杖图标）
2. 从预设中选择：
   - **摘要**：浓缩为3行或更少
   - **商务格式**：转换为正式文本
   - **休闲化**：使其更友好
   - **详细展开**：更具体地阐述
   - **项目符号**：组织以提高可读性
   - **翻译**：转换语言
3. 或输入自定义提示进行自定义指令
4. 查看结果并点击"应用"或"丢弃"

**注意**：需要Claude API密钥。

### 分类摘要

AI分析分类中的所有备忘录并生成摘要。

1. 在分类视图中点击摘要按钮
2. AI分析所有备忘录
3. 显示：
   - 整体摘要
   - 要点
   - 趋势
   - 统计信息

**注意**：需要Claude API密钥。需要3个以上的备忘录。

### AI使用统计

1. 设置 > AI功能 > 使用统计
2. 查看本月的请求数、令牌数和预估成本
3. 按功能和提供商显示使用情况
4. 可导出历史记录或重置

**注意**：API费用直接支付给各提供商。

1. 打开设置

## Apple Watch集成

### 在Apple Watch上创建备忘录

1. 在Apple Watch上打开QuickMemo应用
2. 点击"新建备忘录"
3. 使用语音输入或涂鸦（手写）
4. 选择分类
5. 点击"保存"

### 关于同步

- 备忘录自动与iPhone同步
- 离线时保存在本地，连接时同步
- 最新20条备忘录存储在Watch上

### 检查Pro状态

- Pro购买状态在Watch应用启动时自动更新
- 在iPhone上购买后，重启Watch应用

## 小组件

### 添加小组件

1. 长按主屏幕
2. 点击左上角的"+"按钮
3. 选择"QuickMemo"
4. 选择小组件大小
   - **小**：显示1个分类的备忘录
   - **中**：显示2个分类的备忘录
   - **大**：显示最多8个分类的备忘录

### 自定义小组件（Pro）

1. 设置 > 小组件设置
2. 选择要显示的分类（最多8个）
3. 点击"保存"

**注意**：免费版只显示当前选择的分类。

## 数据管理

### 导出备忘录

#### 导出单个备忘录
1. 在备忘录编辑界面点击共享按钮
2. 选择格式：
   - Markdown (.md)
   - 纯文本 (.txt)
   - JSON (.json)
3. 选择保存位置

#### 批量导出
1. 设置 > 导出数据
2. 选择格式：
   - CSV：可用Excel等打开
   - JSON：结构化数据
3. 导出所有备忘录

### 导入备忘录

支持格式：
- 纯文本 (.txt)
- Markdown (.md)
- CSV（QuickMemo导出格式）
- JSON（QuickMemo导出格式）

#### 导入步骤
1. 设置 > 导入备忘录
2. 点击"选择文件"
3. 选择要导入的文件
4. 在预览中查看内容
5. 选择目标分类
6. 点击"导入"

**注意**：
- 自动检测字符编码（UTF-8、Shift-JIS、EUC-JP等）
- 对于没有分类信息的备忘录，应用选择的分类
- 免费版：100条限制。Pro：无限制。

### iCloud备份（Pro）

#### 创建备份
1. 设置 > iCloud备份
2. 点击"立即备份"
3. 完成后显示最后备份日期

#### 从备份恢复
1. 设置 > iCloud备份
2. 点击"从iCloud恢复"
3. 在确认界面选择"恢复"
4. 备份数据与当前数据合并

**注意**：必须登录iCloud。

## 设置

### 应用设置

#### 摇动手势
- **开/关**：启用/禁用摇动打开备忘录输入
- **灵敏度**：调整摇动力度（3个级别）

#### 通知设置
- **记忆提醒**（Pro）：定期通知检查备忘录
- **免打扰时间**：设置停止通知的时间段

#### 语言设置
- **应用语言**：选择日语、英语或简体中文
- **分类名称**：语言更改时自动翻译

### 数据和隐私

- 备忘录数据安全存储在设备上
- AI API密钥加密存储在钥匙串中
- 仅在使用功能时向AI提供商发送数据
- 隐私政策：https://yok1012.github.io/quickMemoPrivacypolicy/

## 常见问题

### 问1：摇动手势无响应

**答**：请检查以下内容：
- 设置 > 摇动手势已打开
- 尝试将灵敏度更改为"高"
- 尝试轻轻摇动iPhone 2-3次

### 问2：Apple Watch不同步

**答**：请尝试以下操作：
- 验证iPhone和Watch通过蓝牙连接
- 在两个设备上重启QuickMemo应用
- 检查iPhone设置 > 蓝牙，确认Watch已连接

### 问3：已购买Pro但功能不可用

**答**：
1. 完全退出并重启应用
2. 在设置中检查Pro状态
3. 如果仍不起作用，尝试"恢复购买"

### 问4：AI功能不工作

**答**：
- 验证API密钥配置正确
- 检查所需功能的提供商API密钥是否已设置
  - 标签提取：Gemini
  - 备忘录整理/摘要：Claude
- 检查API密钥有效期和使用限制

### 问5：备忘录消失了

**答**：
- Pro版本：设置 > iCloud备份 > 从iCloud恢复
- 检查是否已归档（设置 > 检查归档）

### 问6：删除分类会删除备忘录吗？

**答**：不会，备忘录不会被删除。删除分类时，该分类中的备忘录会自动移至"其他"分类。

### 问7：无法更改小组件分类

**答**：小组件自定义是Pro专属功能。免费版只显示当前选择的分类。

## 故障排除

### 应用无法启动

1. 完全退出应用（双击主页按钮或从底部向上滑动）
2. 重启iPhone
3. 检查App Store是否有更新
4. 如果未解决，重新安装（数据将保留）

### 数据未显示

1. 设置 > 分类管理，验证显示正确的分类
2. 尝试使用搜索栏搜索备忘录
3. Pro版本：尝试从iCloud备份恢复

### 日历集成不工作

1. iPhone设置 > 隐私 > 日历
2. 验证QuickMemo访问已启用
3. 应用设置 > 日历集成已打开
4. 检查日历应用中是否创建了"Quick Memo"日历

### 购买未反映

1. 验证在App Store中使用相同的Apple ID登录
2. 尝试应用内的"恢复购买"按钮
3. 重启iPhone
4. 如果24小时内未反映，联系Apple支持

### AI功能错误

#### "API密钥无效"错误
- 尝试重新输入API密钥
- 在提供商的控制台中验证API密钥有效
- 检查API密钥访问权限

#### "超出请求限制"错误
- 可能已超过提供商的免费额度
- 设置 > AI功能 > 使用统计，检查使用量
- 在提供商的控制台中验证限制和费用

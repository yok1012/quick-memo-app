### サンプルメモ

#### 短文（<50字）
```
明日の会議で新プロジェクトの提案をする
```

#### 中文（50-200字）
```
新しいアプリのUI/UXデザインについて検討。
ユーザーフィードバックを元に、ナビゲーションを改善する必要がある。
特に、設定画面へのアクセスが分かりにくいという指摘が多かった。
次のスプリントで優先的に対応する。
```

#### 長文（>200字）
```
今日のミーティングでは、来期のマーケティング戦略について議論した。
主なポイントは以下の通り：

1. SNS広告の強化
   - Instagram、Twitterでのキャンペーン展開
   - インフルエンサーとのコラボレーション検討

2. コンテンツマーケティング
   - ブログ記事の定期的な更新（週2回）
   - 事例紹介の動画コンテンツ制作

3. SEO対策
   - キーワード分析の実施
   - サイト構造の改善

予算は前期比120%で承認された。
次回のミーティングで具体的なアクションプランを決定する。
```

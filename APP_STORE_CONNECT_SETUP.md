# App Store Connect 設定ガイド

## ガイドライン2.1対応 - アプリ内購入製品の送信

### 必須作業
App Store Connectでアプリ内購入製品を作成・送信する必要があります。

### 手順

#### 1. App Store Connectにログイン
https://appstoreconnect.apple.com

#### 2. アプリ内購入製品の作成

1. **マイApp** → **quickMemoApp** を選択
2. **機能** タブ → **アプリ内購入** を選択
3. **作成** ボタンをクリック
4. **自動更新サブスクリプション** を選択

#### 3. サブスクリプション製品の設定

**製品情報**:
- **参照名**: QuickMemo Pro月額プラン
- **製品ID**: `com.yokAppDev.quickMemoApp.pro.month`（コードと一致させる）
- **サブスクリプション期間**: 1か月
- **価格**: ¥300（または適切な価格帯を選択）

**ローカライゼーション情報**:

**日本語**:
- **表示名**: QuickMemo Pro
- **説明**:
  ```
  QuickMemo Proで全機能をアンロック！

  • カテゴリ無制限作成
  • Apple Watch同期
  • ウィジェット機能
  • 定期リマインダー通知
  • タグ管理機能
  • エクスポート機能
  ```

**英語**:
- **Display Name**: QuickMemo Pro
- **Description**:
  ```
  Unlock all features with QuickMemo Pro!

  • Unlimited categories
  • Apple Watch sync
  • Widget functionality
  • Scheduled reminder notifications
  • Tag management
  • Export features
  ```

**中国語（簡体字）**:
- **显示名称**: QuickMemo Pro
- **描述**:
  ```
  使用QuickMemo Pro解锁所有功能！

  • 无限制分类
  • Apple Watch同步
  • 小组件功能
  • 定时提醒通知
  • 标签管理
  • 导出功能
  ```

#### 4. App Reviewスクリーンショットの追加

**必須**: App Reviewチーム用のスクリーンショットを追加

1. **App Review情報** セクション
2. **スクリーンショットを追加**
3. 購入画面のスクリーンショットをアップロード
   - iPhone 6.5インチ（必須）
   - その他のサイズ（オプション）

**スクリーンショット内容**:
- 購入画面全体
- 価格が表示されている状態
- 機能説明が見える状態

#### 5. サブスクリプショングループの設定

1. **サブスクリプショングループを作成**
2. **グループ参照名**: QuickMemo Proプラン
3. **ローカライゼーション**: 各言語で適切な名前を設定

#### 6. 審査メモの追加

**審査メモ**:
```
このサブスクリプションはQuickMemo Proの全機能をアンロックします。

テストアカウント情報:
- Sandboxテスター: [メールアドレス]
- パスワード: [パスワード]

主な機能:
1. 無制限のカテゴリ作成（無料版は5個まで）
2. Apple Watch同期
3. ウィジェット機能
4. リマインダー通知
5. 高度なタグ管理

クロスデバイスアクセス:
- Sign in with Apple（オプション）で異なるApple ID間でもサブスクリプションを共有可能
- CloudKit経由で同期
```

#### 7. ステータス確認と送信

1. すべての必須項目が入力されていることを確認
2. **保存** をクリック
3. **審査に提出** をクリック

### 重要な注意点

1. **製品IDの一致**
   - コード内: `com.yokAppDev.quickMemoApp.pro.month`
   - App Store Connect: 完全に同じIDを使用

2. **価格設定**
   - 各国の価格帯を確認
   - 自動価格調整を有効化

3. **無料トライアル（オプション）**
   - 必要に応じて設定可能
   - コード側での対応も必要

4. **プロモーションオファー（オプション）**
   - 後から追加可能

### トラブルシューティング

#### 製品が表示されない場合
1. 契約、税金、銀行情報が完了しているか確認
2. アプリ内購入の有効化を確認
3. 製品IDがコードと完全に一致しているか確認

#### 審査で却下された場合
1. スクリーンショットが適切か確認
2. 説明文が機能と一致しているか確認
3. 価格が適切か確認
